# laragccbackend

Laravel API backend for **Lara Beauty Store GCC**.

## EasyPanel deployment

Environment variables for EasyPanel are in:

- **`easypanel.env`** — copy-paste into App → Environment
- **`docs/EASYPANEL.md`** — setup steps (MySQL, domain, mounts)

Generate `APP_KEY` after first deploy: `php artisan key:generate --show`
